### Animatronic Eye Control 仿生眼球控制系统说明文档

#### 一、项目简介

这是一个基于 **Arduino UNO** 与 **Sensor Shield V5.0** 的仿生动画眼球控制系统。通过六个舵机协同工作，分别控制眼球的**左右/上下转动**以及**上下眼睑的张合**。

该程序预设了 1 至 9 的数字快捷命令。你只需在电脑的串口监视器中输入对应数字，即可让眼球执行直视、向特定方向看、眨眼、随机扫视等动作，非常适合电子玩偶、机器人头部原型制作或特效道具开发。

#### 二、硬件零件清单

请根据以下列表准备硬件：

| 序号 | 零件名称 | 数量 | 备注 |
| :-- | :-- | :-- | :-- |
| 1 | **Arduino UNO R3** | 1 块 | 主控板 |
| 2 | **Sensor Shield V5.0** | 1 块 | 扩展板，方便连接多个舵机 |
| 3 | **9g 微型舵机** | 6 个 | 用于控制眼球上下、左右及上下眼睑 |
| 4 | **机械结构件** | 1 套 | 眼球模型支架、连杆、拉杆等（需自行准备） |
| 5 | **USB 数据线** | 1 条 | 用于上传代码及串口通讯供电 |
| 6 | **外部电源（推荐）** | 1 个 | **建议 5V 2A 以上电源**<br>注意：6 个舵机同时动作时电流较大，仅靠 USB 供电可能导致 Arduino 重启或舵机无力 |

#### 三、接线说明

将六个舵机的信号线（通常为黄色或白色线）分别连接到 Sensor Shield 的对应数字引脚上，**电源线（红/黑）需插在扩展板的 5V/GND 排针上**。

> **代码定义引脚对照表：**

| 舵机功能 | 代码变量名 | 对应 Arduino 引脚 |
| :-- | :-- | :-- |
| **眼球左右转 (X轴)** | `servoXAxis` | 9 |
| **眼球上下转 (Y轴)** | `servoYAxis` | 10 |
| **左上眼睑** | `servoEyelidTL` | 11 |
| **右上眼睑** | `servoEyelidTR` | 6 |
| **左下眼睑** | `servoEyelidBL` | 5 |
| **右下眼睑** | `servoEyelidBR` | 3 |

#### 四、操作步骤

**第一步：上传代码**
1. 使用 USB 线将 Arduino UNO 连接至电脑。
2. 打开 Arduino IDE，将提供的代码粘贴到空白窗口中。
3. 选择正确的开发板型号（Arduino Uno）和端口号（COM口）。
4. 点击 **上传** 按钮，等待程序烧录完成。

**第二步：打开串口监视器**
1. 上传成功后，点击 IDE 右上角的 **串口监视器** 图标（放大镜）。
2. 将右下角的 **波特率** 设置为 **9600**。
3. 此时你会看到系统启动提示 `Animatronic Eye Control System` 以及命令菜单。

**第三步：输入命令控制眼球**
在串口监视器顶部的输入框中，输入以下数字并点击 **发送**（或按回车），眼球将做出相应反应：

| 输入数字 | 动作效果描述 |
| :-- | :-- |
| **1** | 正视前方（复位到中间） |
| **2** | 向右看 |
| **3** | 向左看 |
| **4** | 向上看 |
| **5** | 向下看 |
| **6** | 眨眼一次（自动完成闭合-睁开循环） |
| **7** | 眼睛睁大（强制恢复眼睑张开角度） |
| **8** | 眼睛闭合（闭眼约 0.3 秒后自动睁开） |
| **9** | **随机扫视**（眼球自动环顾四周 5 次） |
| **0** | 重新显示当前快捷命令菜单 |

#### 五、机械安装提示

- **舵机归中**：在安装眼球机械连杆前，建议先运行程序，输入数字 `1` 让所有舵机归中至 90 度，然后再安装舵盘和连杆，防止安装错位导致扫齿。
- **电源管理**：如果发现眼球抖动严重或动作卡顿，很可能是供电不足。请务必在 Sensor Shield 的 **外部供电端子** 上接入 5V 稳压电源，切勿让电脑 USB 口承载全部 6 个舵机的电流。
